# Framer-Inspired Design System

## 1. Visual Theme & Atmosphere
Framer’s design is a dark, product-first, cinematic experience.

- Pure black (#000000) background
- Product UI as the hero
- GT Walsheim compressed typography
- Single accent: Framer Blue (#0099ff)

## 2. Color Palette

### Primary
- Black: #000000
- White: #ffffff
- Framer Blue: #0099ff

### Secondary
- Muted Silver: #a6a6a6
- Near Black: #090909

### Surfaces
- Frosted: rgba(255,255,255,0.1)
- Subtle: rgba(255,255,255,0.5)

## 3. Typography

### Fonts
- Display: GT Walsheim
- Body: Inter
- Mono: Azeret Mono

### Sizes
- Hero: 110px
- Section: 62px
- Body: 15px
- Caption: 12px

## 4. Components

### Buttons
- Pill shape (40px–100px radius)
- White CTA or frosted glass
- Ghost for minimal actions

### Cards
- Dark surfaces (#090909)
- Blue ring border glow
- Rounded corners (10–15px)

## 5. Layout

- 8px spacing system
- Large vertical spacing (80–120px)
- Asymmetric layouts (40/60 split)

## 6. Depth

- Blue ring shadows
- Subtle highlights + ambient shadows
- No heavy elevation

## 7. Do's & Don'ts

### Do
- Use pure black background
- Apply negative tracking in headings
- Use pill buttons

### Don't
- Add extra colors
- Use warm blacks
- Use square buttons

## 8. Responsive

- Mobile: stacked layout
- Tablet: 2-column
- Desktop: full layout
