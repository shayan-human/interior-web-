# Ferrari-Inspired Design System

## 1. Visual Theme & Atmosphere
Ferrari's website is a digital editorial — a curated magazine where the brand is presented with artistic gravitas and precision.

- Chiaroscuro layout (black vs white sections)
- Minimal use of Ferrari Red (#DA291C)
- Editorial storytelling through imagery
- Clean, sharp UI with minimal decoration

## 2. Color Palette

### Primary
- Ferrari Red: #DA291C
- White: #FFFFFF

### Dark & Neutral
- Black: #000000
- Dark Surface: #303030
- Near Black: #181818
- Gray Scale: #666666, #8F8F8F, #D2D2D2

### Accent
- Yellow: #FFF200
- Modena Yellow: #F6E500

## 3. Typography

### Fonts
- FerrariSans (Primary)
- Body-Font (Labels, uppercase)

### Key Sizes
- Headings: 24–26px
- Body: 16px
- Labels: 12px uppercase

## 4. Components

### Buttons
- Primary: White bg, black text
- CTA: Ferrari Red bg, white text
- Ghost: Transparent with white border

### Cards
- White background
- No shadows
- Image + text layout

## 5. Layout

- 8px spacing system
- Large vertical spacing (40–80px)
- Alternating dark/light sections

## 6. Depth

- Flat design
- No shadows
- Depth via contrast & imagery

## 7. Do's & Don'ts

### Do
- Use red sparingly
- Keep sharp corners (2px)
- Use high-quality images

### Don't
- Overuse colors
- Add heavy shadows
- Use rounded UI

## 8. Responsive

- Mobile: single column
- Tablet: 2-column
- Desktop: full layout
