# Apple-Inspired Design System

## 1. Visual Theme & Atmosphere
Apple’s design is minimal, cinematic, and product-first.

- Pure black and light gray alternating sections
- SF Pro typography with optical sizing
- Single accent color: Apple Blue (#0071e3)
- Product-focused layouts with no distractions

## 2. Color Palette

### Primary
- Black: #000000
- Light Gray: #f5f5f7
- Near Black: #1d1d1f

### Interactive
- Apple Blue: #0071e3
- Link Blue: #0066cc
- Bright Blue (dark bg): #2997ff

### Text
- White: #ffffff
- Primary: #1d1d1f
- Secondary: rgba(0,0,0,0.8)
- Tertiary: rgba(0,0,0,0.48)

## 3. Typography

### Fonts
- SF Pro Display (headings)
- SF Pro Text (body)

### Sizes
- Hero: 56px
- Section: 40px
- Body: 17px
- Caption: 14px

## 4. Components

### Buttons
- Primary: Blue background, white text
- Dark: Near black background
- Pill CTA: 980px radius outline/filled

### Cards
- Background: #f5f5f7 or dark variants
- Radius: 5–8px
- Shadow: soft only when needed

## 5. Layout

- 8px spacing system
- Full-width sections
- Centered content (~980px max)
- Alternating dark/light sections

## 6. Depth

- Minimal shadows
- Glass navigation blur effect
- Depth via contrast, not layers

## 7. Do's & Don'ts

### Do
- Use Apple Blue only for interactions
- Keep typography tight (negative tracking)
- Use clean product imagery

### Don't
- Add extra colors
- Use heavy shadows
- Use gradients or textures

## 8. Responsive

- Mobile: single column
- Tablet: 2 columns
- Desktop: full layout
